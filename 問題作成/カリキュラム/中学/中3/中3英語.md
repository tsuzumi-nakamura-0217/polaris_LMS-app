# 中学校3年生 英語カリキュラム（学習指導要領準拠）

## ■ねらい（学習指導要領に則した目標）
- 中学校で学習する文法事項（現在完了形、後置修飾・関係代名詞、間接疑問文など）を完成させ、「聞く」「話す」「読む」「書く」の4技能をさらに高め、実践的に活用できる力を養う。
- 社会的な話題や世界平和、環境問題など多様なテーマについて、自分の考えや理由を持ったまとまりのある文章で表現したり、議論したりできる力を育てる。
- 3年間の学習の集大成として、将来の進路や世界に目を向けたコミュニケーションの基礎を確立する。

---

## ■構成（カテゴリ＝単元）
※本カリキュラムは Polaris LMS の「英語」科目で利用するカテゴリ名（単元名）として想定しています。

### 1. **受動態の復習と発展**
- `be + 過去分詞` の基本構造の復習
- 助動詞を含む受動態（`will be + 過去分詞`, `can be + 過去分詞` など）
- 第4文型（SVOO）の受動態、第5文型（SVOC）の受動態的な表現

### 2. **現在完了形（経験・継続・完了・結果）**
- **経験用法**: `have/has + 過去分詞` を用いた経験の表現（ever, never, once, ~ times など）
- **継続用法**: 過去から現在まで続いている状態の表現（for ~, since ~ など）
- **完了・結果用法**: 動作の完了やその結果の表現（just, already, yet など）
- 現在完了進行形（`have/has been + ~ing`）：動作の継続（※現行の学習指導要領で中3への移行事項）

### 3. **不定詞の発展・原形不定詞**
- `疑問詞 + to + 動詞の原形`（how to ~, what to ~, where to ~ など）
- `It is ~ (for 人) to ...` の構文の定着と応用
- 原形不定詞（使役動詞 `make / let / have` + O + 動詞の原形、知覚動詞 `see / hear` など）※現在の中学英語では基礎的な理解に留める場合が多いが、新指導要領で一部拡張

### 4. **分詞による後置修飾（形容詞的用法）**
- **現在分詞（~ing）**: 名詞を後ろから修飾し、「〜している（名詞）」を表す（the boy running in the park）
- **過去分詞**: 名詞を後ろから修飾し、「〜された（名詞）」を表す（the book written by him）

### 5. **関係代名詞（主格）**
- 人を表す主格の関係代名詞 `who`（the boy who is running...）
- 事物を表す主格の関係代名詞 `which`（the book which is on the table...）
- 人・事物の両方に使える主格の関係代名詞 `that`

### 6. **関係代名詞（目的格）**
- 人・事物を表す目的格の関係代名詞 `which / that / (who/whom)`（the book which I bought yesterday...）
- 目的格の関係代名詞の省略

### 7. **間接疑問文**
- 疑問詞で始まる疑問文を文の一部に組み込む表現（`I know what he likes.` / `Do you know where she lives?` など）
- 平叙文の語順（疑問詞 ＋ 主語 ＋ 動詞）になる規則

### 8. **仮定法（仮定法過去）**
- 現在の事実に反する仮定や願望の表現（※現行の学習指導要領で中3の学習事項）
- `If + S + 動詞の過去形(were/did), S + would/could + 動詞の原形`（If I were you, I would...）
- `I wish + S + 動詞の過去形`（I wish I could...）

> ※上記の内容は、**小学・中学1・2年生の内容（現在形、過去形、進行形、未来表現、助動詞、基本的な接続詞・不定詞・動名詞、比較など ※復習を除く）を含まず、高校生の内容（仮定法過去完了、過去完了形、関係副詞など）を含まないように配列**されています。
> ※日本の現行の学習指導要領（現在完了進行形や仮定法過去を含む）における標準的な中学3年生の教科書の配列に準拠しています。
