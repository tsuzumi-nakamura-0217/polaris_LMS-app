# 中学校2年生 英語カリキュラム（学習指導要領準拠）

## ■ねらい（学習指導要領に則した目標）
- 中学1年生で学習した基礎（現在形、過去形、進行形など）の上に、より多様で複雑な表現や文法事項を習得し、「聞く」「話す」「読む」「書く」の4技能を総合的に伸ばす。
- 時制の幅（未来形など）や、様々な状況に応じた表現（助動詞、接続詞、不定詞、動名詞、比較、受動態）を学び、自分の意見や理由、経験などをより詳細に英語で伝えられるようにする。
- 異文化理解を深めるとともに、日本や地域の事象についても英語で紹介する基礎を築く。

---

## ■構成（カテゴリ＝単元）
※本カリキュラムは Polaris LMS の「英語」科目で利用するカテゴリ名（単元名）として想定しています。

### 1. **未来の表現（be going to / will）**
- `be going to ~` を用いた未来の予定や意図を表す肯定文・否定文・疑問文
- 助動詞 `will` を用いた未来の予測や意志を表す肯定文・否定文・疑問文
- 未来を表す語句（tomorrow, next week, in the future など）との呼応

### 2. **助動詞の拡充（must / may / should / have to など）**
- `must` と `must not` （義務・禁止）
- `have to` と `don't have to` （必要・不必要）および `must` との違い
- `may` （許可・推量）
- `should` （助言・当然）

### 3. **接続詞（that / because / when / if など）**
- 従位接続詞 `that` を用いた名詞節（I think that ~ / I know that ~ など）
- 理由を表す `because`
- 時を表す `when` （〜するとき）
- 条件を表す `if` （もし〜ならば）

### 4. **不定詞（名詞的用法・形容詞的用法・副詞的用法）**
- **名詞的用法**: `to + 動詞の原形` が主語や目的語・補語になる働き（I want to ~ / To learn English is ~ など）
- **形容詞的用法**: 名詞を後ろから修飾する働き（something to drink / time to go など）
- **副詞的用法**: 目的・原因・結果を表す働き（I went to the park to play soccer. / I am glad to see you. など）
- `It is ~ (for 人) to ...` の構文

### 5. **動名詞（V-ing）**
- `動詞のing形` が名詞の働きをする用法（主語・目的語・補語・前置詞の目的語）
- 動名詞（V-ing）と不定詞（to V）の使い分け（enjoy, finish, stop など動名詞のみを目的語にとる動詞、want, hope, decide など不定詞のみをとる動詞）

### 6. **比較（原級・比較級・最上級）**
- **原級**: `as ~ as ...` や `not as ~ as ...` による同等比較
- **比較級**: `~er than ...` や `more ~ than ...` による比較
- **最上級**: `the ~est (in/of ...)` や `the most ~ (in/of ...)` による最上級の表現
- 不規則変化をする形容詞・副詞（good/well - better - best, bad/ill - worse - worst など）

### 7. **受動態（be動詞 + 過去分詞）**
- 受動態の基本構造（S is/are/was/were + 過去分詞 + by ~）
- 能動態から受動態への書き換え
- 過去形の受動態
- 助動詞を含む受動態（will be + 過去分詞 / can be + 過去分詞）

### 8. **基本文型の整理（SVC, SVO, SVOO, SVOC）**
- 第2文型（SVC）: look, sound, become など
- 第4文型（SVOO）: give, show, teach, tell, buy などの、目的語を2つとる動詞の用法と、第3文型（SVO + to/for）への書き換え
- 第5文型（SVOC）: make O C, call O C, keep O C などの用法

> ※上記の内容は、**小学・中学1年生の内容（復習を除く）や 中学3年生以上（現在完了形、関係代名詞など）の内容を含まないように配列**されています。
> ※日本の学習指導要領における標準的な中学2年生の教科書の配列（未来形 → 助動詞 → 接続詞 → 不定詞・動名詞 → 比較 → 受動態など）に準拠しています。
